# 6주차 실습 기록
## 사용한 에셋
- 이미지: (Hole.png, [출처](https://www.pngall.com/hole-png/download/36973/))
- 사운드: (eating_sound_effect.mp3, [출처](https://pixabay.com/sound-effects/people-eating-sound-effect-36186/))
## 사용한 AI 프롬프트 (요약)
1. 효과음을 재생 중일 때 다시 누르면 처음부터 재생되게 해줘
2. 특정 행동을 할 때마다 효과음을 출력해줘
## AI 답변에서 도움이 된 것
1.convert_alpha()를 사용해야 성능과 투명도가 좋아짐을 알게 됨.
2.fade_ms를 활용해 배경음악을 자연스럽게 시작하는 방법을 알게 됨.
3.MOUSEDOWN 이벤트를 통해 마우스 클릭을 인식시킬 수 있게 함.
## AI 답변을 수정하거나 버린 것
1.화면 크기에 맞게 좌표 조건(800, 600)을 상황에 맞게 수정함
2.속도가 너무 빨라서 speed 값을 줄여서 조정함
3.효과음이 너무 크게 들려서 볼륨을 따로 조절함
## 적용 결과
- 잘 된 것:
  사운드 기능이 정상 작동
  이미지 이동 및 화면 처리 로직 구현 성공
  여러 기능을 하나의 코드에 통합하는 과정
- 어려웠던 것:
  배경음악과 효과음 관련 함수 구분
- 다음에 시도할 것:
  여러 개의 이미지(오브젝트) 동시에 움직이기
  충돌 판정(캐릭터끼리 부딪힘) 구현
